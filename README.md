# Affirmations App

Aplikasi Android menampilkan daftar afirmasi positif menggunakan **LazyColumn** dan **Card** dengan Jetpack Compose.

Berdasarkan Android Codelab: *Membuat Daftar yang Dapat Di-scroll* (Affirmations App).
